## [0.1.0] - 2024-10-30

### Added
- Initial release of Kunal Dark Dev Theme
- High-contrast dark color scheme with vibrant syntax highlighting
- Deep black (`#000000`) and dark blue (`#040616`) backgrounds
- Vibrant accent colors:
  - Pink (`#f90086`) for keywords and control flow
  - Cyan (`#54d4ff`, `#00e5ff`) for functions and variables
  - Yellow (`#FFF900`) for strings and constants
  - Purple (`#7a6deef4`) for classes and types
- Semantic highlighting support enabled
- Optimized UI colors for:
  - Activity Bar
  - Sidebar
  - Status Bar
  - Editor tabs
  - Terminal
  - Git decorations
  - Peek views and widgets
- Multi-language syntax support (JavaScript, TypeScript, Python, PHP, Java, Go, Rust, and more)
- Comprehensive README with installation and customization guide
- Demo screenshots (`demo.png`, `theme.png`)
- MIT License

### Features
- Eye-strain reduction for long coding sessions
- Clear visual hierarchy for better code readability
- Cohesive workbench design
- Git integration with distinct colors for added, modified, and deleted files
- Custom terminal ANSI colors matching theme palette
